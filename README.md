# Sistema-CDP
Um sistema de Controle de Despesas Pessoais.

## Objetivo
O intuito principal é criar uma ferramenta de fácil usabilidade para organização financeira em nível doméstico, com metas e controle de gastos.
Além disso, o CPD busca facilitar a rastreabilidade dos gastos ao longo do mês para proporcionar melhores estratégias de economia pessoal.
